Welcome to barq-mind

A 30-second tour of the prototype.

  1. Load Model downloads LFM2.5-1.2B-Instruct (~700 MB on first run, cached after).
  2. Sample ingests a synthetic policy brief and triggers tree summarization.
  3. Type a question (try: "What is the implementation timeline?") and watch the navigation path in the right pane.
  4. Toggle show trace in the toolbar to see why the agent chose each step.
